ARQUITECTURA DE CONTEXTOS
AuthContext (useAuth)
Expone: user, firebaseUser, loading, login, logout, register
Fuente: Firebase Auth + Firestore (users/{uid})
Roles válidos: ['admin', 'auditor', 'gestor', 'lector', 'operador']

AppContext (useAppContext)
Expone: sites, questionnaires, audits, isAuthenticated, isAdmin, isAuditor, isGestor, isLector, selectedSector, setSelectedSector, validateAuditRequirements
Consume useAuth() internamente
Filtra sitios por user.assignedSites (no-admin)
Filtra cuestionarios por user.assignedQuestionnaires (no-admin, desde Firestore)

REGLAS DE FIREBASE
javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read: if request.auth != null;
      allow create: if request.auth != null && request.auth.uid == userId && request.resource.data.role in ['lector', 'auditor', 'gestor', 'operador'];
      allow update, delete: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /sites/{document} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /audits/{document} {
      allow read: if request.auth != null;
      allow create: if request.auth != null && request.resource.data.auditorId == request.auth.uid;
      allow update, delete: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /questionnaires/{document} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /sectors/{document} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /notifications/{document} {
      allow read, create: if request.auth != null;
      allow update: if request.auth != null;
      allow delete: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /userSiteAssignments/{document} {
      allow read, write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /auditSchedules/{document} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /solicitudes/{document} {
      allow read: if request.auth != null;
      allow create: if request.auth != null;
      allow update: if request.auth != null && (resource.data.solicitanteId == request.auth.uid || get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin');
      allow delete: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /requisitos/{document} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    match /{document=**} {
      allow read, write: if false;
    }
  }
}

FUNCIONES DE CLASIFICACIÓN (auditUtils.ts)
Auditoría BPM:
text
criticosNC >= 2  → riesgo_alto
score >= 90      → conforme
score >= 81      → a_mejorar
score < 81       → riesgo_alto
Checklist Municipal:
text
criticosNC >= 1  → riesgo_alto (posible clausura)
score < 70       → riesgo_alto
score < 85       → a_mejorar
score >= 85      → conforme
